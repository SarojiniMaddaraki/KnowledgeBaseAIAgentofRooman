## ğŸ“ Architecture

![Architecture Diagram](assets/architecture-diagram.png)

### RAG Pipeline Flow

1. **User Input**: User asks a question through Streamlit UI
2. **Query Embedding**: Question is converted to 768-dimensional vector using SentenceTransformer
3. **Vector Search**: Pinecone retrieves top-5 most relevant document chunks
4. **Context Assembly**: Retrieved chunks are combined with metadata
5. **LLM Generation**: Gemini generates answer based on retrieved context
6. **Response Display**: Answer is shown with source citations

## âš¡ Performance Optimization

### Current Settings
- **Top-K Retrieval**: 5 chunks
- **Max Tokens**: 1000 (LLM response)
- **Embedding Dimensions**: 768

### Optimization Tips
1. **Increase Top-K** (3-10) for more context
2. **Adjust chunk size** in ingestion for better retrieval
3. **Cache embeddings** using `@st.cache_resource`
4. **Use smaller models** for faster response (trade-off: accuracy)
